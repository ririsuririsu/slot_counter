# スマスロ 甲鉄城のカバネリ 海門決戦 — カウンター仕様

本アプリの「カバネリ海門決戦」機種で使用する確率データと設定推測ロジックの根拠。

## カウント対象

### ① チャンス目 成立/発光カウント

3種のチャンス目を成立毎にカウントし、成立時にアイコンが発光した場合は発光もカウントする。
合算発光率から設定を推測する。

| チャンス目 | 出現率（全設定共通・参考） |
|---|---|
| 無名チャンス目 | 1/38.9 |
| 生駒チャンス目 | 1/44.5 |
| カバネチャンス目 | 1/77.7 |
| 単独チャンス目合算 | 1/16.4 |

**カウント条件（重要）**: 発光率の設定差は「カバネリ高確以外」かつ「非発光中」の
**単独チャンス目**成立時のみで確認されている。以下は発光カウントの対象外:

- カバネリ高確中のチャンス目
- アイコン発光中（前兆中）のチャンス目
- チャンス目高確中に引いた最初のチャンス目

### ② 下段ベルカウント

通常時・AT中問わず出現する下段ベルをカウントし、総ゲーム数に対する出現率から設定推測する。

## 設定差データ

### 下段ベル確率（解析値）

| 設定 | 確率 |
|---|---|
| 1 | 1/121.1 |
| 2 | 1/114.4 |
| 3 | 1/112.8 |
| 4 | 1/106.2 |
| 5 | 1/104.2 |
| 6 | 1/99.1 |

目安: 2000G時点で 1/105 を上回っていれば高設定に期待。

### チャンス目アイコン発光率（実戦値ベースの参考値）

**公式解析値は未公表**。実戦値として 設定1 約7〜12%、設定6 約15〜21% が報告されており、
本アプリでは以下の補間値を尤度計算に使用する（[src/data/kabaneriDefinitions.ts](../src/data/kabaneriDefinitions.ts) の `flashRates`）:

| 設定 | 発光率（参考値） |
|---|---|
| 1 | 10.0% |
| 2 | 11.0% |
| 3 | 12.0% |
| 4 | 13.5% |
| 5 | 15.0% |
| 6 | 17.0% |

終日で約21%の発光率なら設定6に強い期待。参考値のため、発光率による推測結果は
あくまで目安として扱い、より信頼できる解析値が出たら `flashRates` を更新すること。

## 推測ロジック

[src/utils/kabaneriEstimation.ts](../src/utils/kabaneriEstimation.ts):

- 下段ベル: 二項分布の尤度によるベイズ推定（事前確率均等）
- 発光率: 同上（3種チャンス目の合算発光数 / 合算成立数）
- 総合: 両者の尤度を独立として掛け合わせて正規化
