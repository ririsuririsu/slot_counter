## あべしシステム

### 基本仕組み
- 毎ゲーム最低 **1あべし** 加算
- 天破の刻中は大量あべし獲得（5~999あべし/G）
- 実質的に 1G あたり約2あべし増加
- 規定あべし到達で **AT当選**
- 規定あべしを超えた分は **次回へ持ち越し**

### 天井恩恵
- 最大: 1,536あべし+α（設定変更後は1,280あべし+α）
- 1,473あべし超えでのAT当選時は **ATレベル3以上濃厚**（高出玉性能）

## シャッター演出（規定あべし示唆）

### 概要

通常時、特定あべし数到達時に「上からシャッターが閉まる演出」が発生することがある。
シャッターが閉まった場合 → **896あべし以内でのAT当選が濃厚**（＝モードB以上の可能性が非常に高い）。

※ モードB確定ではなく「896あべし以内示唆」。モードAでも896以内に規定あべしがあれば発生しうる。

### シャッター判別ポイント（5箇所）

| # | チェックあべし帯 | シャッター発生時の意味 | 備考 |
|---|----------------|---------------------|------|
| 1 | **37~45あべし** | 896あべし以内濃厚 | 最速判別ポイント |
| 2 | **166~173あべし** | 896あべし以内濃厚 | 第2判別ポイント |
| 3 | **288~300あべし** | **判別不可** | 全モードで発生するため使えない |
| 4 | **421~429あべし** | 896あべし以内濃厚 | |
| 5 | **489~495あべし** | 896あべし以内濃厚 | |

### シャッター判別の実践法則

- 第1・第2ポイント（37~45、166~173）の **両方でシャッターが来ない** → 1,000あべし超えがほぼ確定（モードAの深いゾーン濃厚）
- 第3ポイント（288~300）は全モードで発生するため **単体では判別に使えない**
- シャッターの有無と天命の刻（前兆）突入は無関係。シャッターが閉まること自体が示唆

### シャッター狙いの立ち回り

1. 37~45あべし手前の台を確保
2. シャッターが閉まるか確認
3. 閉まれば896あべしまで続行
4. 閉まらなければ166~173あべしまで回して再確認
5. それでも閉まらなければやめ（モードAの深いゾーン濃厚）

## 前兆システム

### 天命の刻（前兆ステージ）
- 規定あべし到達時に突入する前兆ステージ
- 「南斗の居城ステージ」移行 → **本前兆濃厚**
- ハズレ → **フェイク前兆濃厚**

### フェイク前兆によるモード判別

特定あべし帯でフェイク前兆が発生すると、モードを判別できる。

| フェイク前兆のあべし帯 | 判別内容 | 備考 |
|-----------------------|---------|------|
| 1~64あべし | **モードB以上確定** | |
| 129~192あべし | **モードB or C確定** | |
| 257~320あべし | **モードB or C確定** | |
| 321~384あべし | モードB/Cの可能性アップ | 発生率が他の約5倍と高いためモード確定にはならない |

**注意**: 天破の刻でのあべし加算でゾーンを跨いだ場合、フェイク前兆が発生しないことがあり100%の判別は困難。

## アプリ設計：履歴ログベースの設定推定

### 設計方針

小役確率に設定差がないため、**プレイ履歴のログ**を時系列で蓄積し、モード推移と設定を推定する。

### セッション開始情報

プレイ開始時に記録する。以降の全ログ解釈の前提条件となる。

| フィールド | 説明 |
|-----------|------|
| 朝一リセット判定 | リセット済み / 据え置き / 不明 |

**リセット vs 据え置きの影響:**

| 項目 | リセット時 | 据え置き時 |
|------|-----------|-----------|
| モード抽選 | 設定に応じて再抽選 | 前日のモードを引き継ぐ |
| モードA天井 | **1,280あべしに短縮** | 1,536あべし |
| あべし数 | リセット（+ランダム加算でスタート） | 前日のあべし数を引き継ぐ |
| 256あべし以内選択率 | 設定差あり（設定1: 36.4% vs 設定6: 52.6%） | モード依存 |
| 内部状態 | 低確からスタート | 前日の状態を引き継ぐ |

**リセット判別の手がかり:**
- 朝一の**あべし表示が0付近**ならリセット濃厚（据え置きは前日の値を引き継ぐ）
- 朝一に**シャッター判別**（37~45あべし）を行い、前日のモードと矛盾すればリセット
- **256あべし以内の早い当たり**が多ければリセット＋高設定の可能性

**推定処理への影響:**
- リセット時 → 設定変更時のモード移行率テーブルを使用、天井1,280に短縮
- 据え置き時 → 通常のモード移行率テーブルを使用、天井1,536
- 不明時 → 両方の可能性を加重平均して推定

### 主要な履歴ログの種類

#### 1. AT当選ログ

| フィールド | 説明 |
|-----------|------|
| 当選G数 | AT当選時の総ゲーム数 |
| 当選あべし数 | AT当選時のあべし数 |
| 当選契機 | 規定あべし到達 / レア役直撃 |
| 直撃時の契機役 | 弱チェリー / 強チェリー / チャンス目 / スイカ / 確定チェリー 等 |

→ AT初当たり確率、当選あべし帯からモード推定に使用

#### 2. 天破の刻ログ

| フィールド | 説明 |
|-----------|------|
| 発生G数 | 天破突入時の総ゲーム数 |
| 当選契機 | 弱チェリー / 強チェリー / スイカ / チャンス目 / 勝舞揃い / 確定チェリー |
| 内部状態（推定） | 低確 / 通常 / 高確 / 伝承 |
| 継続G数 | 7G / 14G / 21G / 無限 |

→ 天破出現率の算出に使用

#### 3. 天撃チャレンジログ（天破の刻中 毎ゲーム記録）

| フィールド | 説明 |
|-----------|------|
| ゲーム番号 | 天破中の何G目か |
| 成立役 | ハズレ / リプレイ / ベル / 弱チェリー / 強チェリー / スイカ / チャンス目 / 勝舞揃い / 確定チェリー |
| 天撃結果 | 成功（突破） / 失敗 |

→ 成立役ごとの天撃成功率を算出。特にハズレ時成功率（設定1: 4.7% vs 設定6: 16.4%）が最重要

#### 4. 前兆ログ

| フィールド | 説明 |
|-----------|------|
| 発生あべし数 | 前兆発生時の推定あべし数 |
| 前兆種別 | 本前兆 / フェイク前兆 |
| シャッター有無 | シャッター演出の発生有無 |
| 推定モード | フェイク前兆のあべし帯から推定 |

→ モード判別に使用。シャッター演出との組み合わせで精度向上

#### 5. 演出示唆ログ

| フィールド | 説明 |
|-----------|------|
| 100Gランプ（A箇所） | 色 + 点灯/点滅 → モード・あべし示唆 |
| 100Gランプ（B箇所） | 色 + 点灯/点滅 → 設定示唆 |
| 100Gランプ（C箇所） | 色 + 点灯/点滅 → 初当たり近さ示唆 |
| サミートロフィー | 色 → 設定下限確定 |
| その他（台座色、LED色） | 天国示唆等 |

#### 6. 伝承モードログ

| フィールド | 説明 |
|-----------|------|
| 伝承タイプ | ショート / ミドル / ロング |
| 終了後の状態 | 低確 / 通常 / 高確（ステージで推定） |

→ 終了後の高確移行率（設定1: 3.1% vs 設定6: 10.9%）の算出に使用

### ログから行う推定処理

| 推定項目 | 使用する履歴 | ロジック |
|---------|------------|---------|
| **現在のモード推定** | 前兆ログ（シャッター含む） + AT当選あべし + ランプ示唆 | シャッター判別+フェイク前兆帯+ランプからモードA/B/C/天国を絞り込む |
| **規定あべし推定** | シャッター判別 + モード推定 | シャッター閉=896以内、シャッター未発生=モードA深部の可能性 |
| **設定推定** | AT当選率 + 天破出現率 + 天撃成功率（役別） + 伝承後状態 + 演出確定 | ベイズ推定で各設定の事後確率を算出 |

### 設定判別の優先度（重み付け）

1. **天撃ハズレ時成功率** — 最大3.5倍差、最重要
2. **AT初当たり確率** — 1/300基準
3. **天破の刻突入率** — 1/90基準
4. **伝承モード後の高確移行率** — 設定4以上で大きく変化
5. **モード移行傾向** — B以上の頻度で高設定判別
6. **演出確定系** — トロフィー・ランプで設定下限が判明
